# Bike-Classifier

Mountain Bike and Road Bike Classifier using Tensorflow

### Dependencies
---

#### TensorFlow
> pip install tensorflow

#### OpenCV
> pip install opencv-python

#### scikit-learn
> pip install scikit-learn

### Download Images
---
```sh
$ python download.py
```

### Train
---
```sh
$ python train.py training_images
```

### Test
---
```sh
$ python test.py test_images
```
